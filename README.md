# Sudoku-Generator
Generates a solved Sudoku board!
Created by Yujing Sun

Description: This program is written in Java and generates a random completed 9 by 9 Sudoku board and prints it to the console each time the user runs it. The main goal of this project is to create a valid Sudoku board where each row, column, and 3 by 3 box contains the number 1 through 9 without repeating. 

How to Run:
Compile and run the Java file in a console.

Summary: The program first creates a random first row and then uses a shifting pattern (3,3,1,3,3,1,3,3) to generate the rest of the board that follows Sudoku rules.
